HTML TAGS:
    <html></html> - обгортка для HTML сайту (можна використовувати лише один раз (задля SEO оптимізвції))
    <body></body> - тег що включає весь контент що знаходиться на сторінці
    <p></p> - контейнер для тексту (параграфу)
    <h1></h1> - заголовок першого (найвищого) рівня
    <h2></h2> - заголовок рівня 2
    <h3></h3> - заголовок рівня 3
    <h4></h4> - заголовок рівня 4
    <h5></h5> - заголовок рівня 5
    <h6></h6> - заголовок рівня 6
    <u></u> - підкреслює текст
    <b></b> - виділяє текст товстим
    <i></i> - виділяє текст курсивом
    <sub></sub> - нижній індекс
    <sup></sup> - верхній індекс

    HTML 5 TAGS
    <audio></audio> - тег для вставлення аудіофайлів
    <video></video> - тег для вставлення відеофайлів
    <section></section> - тег для виділення певної частини сайту
    <aside></aside> - тег для виділення бокової частини сайту
    <article></article> - тег для виділення статті або тексту
    <header></header> - тег для виділення верхньої частини сайту
    <main></main> - тег для виділення основної частини сайту
    <footer></footer> - тег для виділення нижньої частини сайту
